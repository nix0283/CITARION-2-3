## API Overview

### REST API
- **Base URL**: `https://www.okx.com`
- **Demo URL**: `https://www.okx.com` (with demo flag)

### Authentication
- `OK-ACCESS-KEY` - API Key
- `OK-ACCESS-SIGN` - Signature
- `OK-ACCESS-TIMESTAMP` - Timestamp
- `OK-ACCESS-PASSPHRASE` - Passphrase
