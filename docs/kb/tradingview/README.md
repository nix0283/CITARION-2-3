## 🎨 Widgets

### Advanced Chart Widget
```html
<script src="https://s3.tradingview.com/tv.js"></script>
<div class="tradingview-widget-container">
  <div id="tradingview_chart"></div>
  <script>
    new TradingView.widget({
      "symbol": "BINANCE:BTCUSDT",
      "interval": "1H",
      "theme": "dark"
    });
  </script>
</div>
```
